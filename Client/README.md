# Client

This is the client side of the application.

# Development

## Requirements

- bun
- typedoc
- webpack
- playwright + (VsCode extension)

## Installation

```bash
bun install
```

## Run

```bash
bun run dev
```

## Build

```bash
bun run wbuild
```

## Test

Execute tests with:

```bash
bun run test
```

Execute tests coverage with:

```bash
bun run cov
```

Execute tests in ui with:

```bash
bun run testui
```

Execute test codegen with:

```bash
bun run codegen
```

## Documentation

```bash
bun run doc
```
